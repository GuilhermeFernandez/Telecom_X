<h1 align="center"> Análise de Churn da TelecomX </h1>
<p align="center"> Projeto desenvolvido a partir do desafio da Alura e da Oracle Next Education. O código conta com analise de variáveis categóricas e outras métricas para entender a desistência de clientes. Também conta com visualização em gráfico das diferentes métricas comparativamente. </p>

<h2> Estrutura do Projeto </h2>
<p> O projeto é estruturado da seguinte maneira:<br>
        1. Leitura dos dados;<br>
        2. Limpeza e tratamento dos dados; <br>
        3. Análise exploratória dos dados;<br>
        4. Visualização dessas informações e outras através de gráficos.<br>
        <br>
    
</p>

<h2> Instruções para uso </h2>
<p> É recomendado que o arquivo enviado tenha o formato JSON e que seja verificado o nome das colunas aninhadas, se necessário o nome das colunas pode ser atualizado ao longo do código.</p>
